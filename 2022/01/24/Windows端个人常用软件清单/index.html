Windows端个人常用软件清单

  • 效率
  • 知识分享

从小学开始就在使用电脑了，使用过的软件也有不少了了！
使用电脑是从用流氓浏览器玩网络小游戏开始，但到现在，使用的软件都追求极致的简洁，尽可能高效！
该文章将介绍我日常所使用的这些软件，如果你对此感兴趣，可下载安装，尝试一下该软件是否适合你；如果有更好的推荐，可在文章底部评论区评论哦！

软件清单

桌面

使用电脑一般都会外接一个大屏的显示器，组成左右两个双屏，桌面一般仅有计算机和回收站两个应用图片，极简！

桌面是自己照一张喜欢的图片作为桌面！

社交

QQ&微信

  • 腾讯家的软件和极简沾不上边，但是这是不得不用的社交软件；
  • 除了腾讯的一系列花里胡哨的没用的功能，QQ的整体上还是很好用的，特别是群文件功能，方便文件的传输和暂存；
  • 微信很不好用，感觉像个半成品，但是不得不用；
  • 设置开机不自启动，每次电脑开机第一件事就是登陆QQ和微信（微信支持不需要手机确认登陆了，谢天谢地！）。

网易邮箱大师

  • 可以满足在一个平台上查看所有的邮件；
  • 新邮件具有提示音，也有气泡通知栏，高效率；
  • 设置开机自启动。

笔记软件

Drawboard PDF

  • UWP软件
  • 阅读与修改PDF文件；
  • 在surface上面使用该软件进行PDF的手写输入，制作电子PDF笔记；

Xodo

  • UWP软件
  • PDF文件的阅读；
  • docx，pptx，xlsx类型阅读时使用该软件，需要对这些类型进行修改时在使用office套件；

Adobe Acrobat DC

  • PDF文件的高阶修改；

Sublime Text

  • txt，c，py，html，css，md等各种文件的阅读和修改，响应速度极快；
  • 深色主题体验很棒！

Tpyora

  • md文件的编辑；
  • 1.0 版本之后的typora需要付费使用了。

Obsidian

  • md文件归档，管理工具；
  • 插件丰富，使用体验好；
  • 但是有点复杂，需要一定的时间去学习使用；

Zotero

  • 文献管理软件，科研必备

OneNote

  • UWP软件
  • 笔记的速记；
  • 前surface笔记软件，但是微软对自己家的软件优化都不好，特别是文件大了时，使用体验不好！
  • 还有就是OneNote的笔记都是以云文件的形式存储在OneDrive中，同步慢，且不是通用文件格式，分享给别人不便；

Xmind

  • 思维导图软件

办公软件

腾讯会议

  • 学生开户用腾讯会议较多

Office套件

  • 办公需要，但日常不喜欢用；

Visio Draw.io

  • 流程图
  • Draw.io是免费软件，体验比Visio要好

PPT模板大全

  • UWP软件
  • 下载一些PPT模板，无广告，模板较多，使用体验好

浏览器

Microsoft Edge

  • 主力浏览器；
  • 插件市场丰富；
  • bing作为主力搜素引擎，yandex辅助搜索引擎；

Google Chrome

  • 辅助浏览器；
  • Google作为搜素引擎；

开发工具

Visual Studio 2019

  • C与C++工程；
  • C与C++小文件的编写使用VSCode+GCC

Visual Studio Code

  • 编写任何文档，与Sublime Text用途一样，只是VSCode启动速度不然SubimeText；
  • 软件的快速打开使用SublimeText，对文件夹下的大量文件的长时间修改时，使用VSCode；
  • 插件市场丰富，nice；

PyCharm

  • Python工程；
  • python小程序使用Sublime Text编写，使用Terminal命令行直接运行；

IntelliJ IDEA

  • HTML前端项目；
  • Java项目；

Matlab

  • Matlab程序的编写与运行；
  • 因为外观不好看，因此有时候会使用VSCode编写程序，Matlab运行程序；

Android Studio

  • Android 项目；

电子设计

Arduino

  • Arduino系列开发板；
  • ESP系列开发板；

Altium Designer

  • 画板子

Proteus 8 Professional

  • 单片机仿真

Multisim

  • 电路仿真

emu8086

  • 8086芯片仿真

Keil

  • 单片机程序编写

Fritizing

  • 电路连接示意图；
  • 示意图主要是当作文章的插图使用；

网盘

百度云

  • 文件的分享与下载；
  • 文件的定期备份；

OneDrive

  • 个人版5GB
  • 学生版1TB
  • 存储OneNote笔记本；
  • 存储Snipaste的截图；
  • 因为OneDrive的速度太慢了，不是主力同步网盘，主要是文件的存储，且为不经常使用的文件；

坚果云

  • 超级好用！！！！
  • 重度使用者建议订阅专业版；
  • 轻度使用免费版够用，每月1GB上传流量限制；

终端

Windows Terminal

  • 相比于cmd和power shell，Terminal是是真的好看！
  • 具有超高的自定义选择，nice！重度使用！

截图工具

Snipaste

  • 也许是Windows客户端最好用的截图工具！
  • 所有的截图我都设置为自动保存到本地，然后再把本地的截图利用OneDrive同步。

翻译

网易有道词典

copytranslator

远程连接

向日葵

  • 三台Windows主机，一台Linux主机，一部手机，我所有设备中均有安装了向日葵；
  • 远程控制，好用！

Xftp

  • 服务器文件传输；

Xshell

  • 服务器远程调用；

图床

PicGo

  • 配置了gitee图床，写markdown文件的好帮手！

虚拟机

WSL

  • WSL比VM速度快太多了，虽然只提供了部分Linux的功能，简单使用足够；
  • 真的使用完整版的Linux时，我会使用向日葵远程调用家里的Linux物理机；

安全

CCleaner

  • 定期清理系统

Sandboxie-Plus

  • 来路不明的文件，在沙盒中打开

火绒安全

  • 清理垃圾；
  • 拦截弹窗；
  • 与Windows Defender联合使用，防病毒；

音视频播放

哔哩哔哩动画UWP

PotPlayer

QQ音乐

网易云音乐

Groove 音乐

音视频制作

Adobe Premiere

Adobe Photoshop

Adobe Lightroom

输入法

微软拼音

  • 简洁；
  • 功能很多，但是学习成本高，需要花费一点时间

录屏

Xbox Game Bar

OBS

解压缩

Bandizip

高效率工具

Everything

  • 快速搜索系统文件

Wox

  • 快截搜索；
  • 插件丰富；
  • 联合everything实现文件的快速搜素；

DeskPins

  • 小工具，将窗口固定在桌面最前端；

EarTrumper

  • UWP软件
  • 独立调节各个软件的音量

QuickLook

  • UWP软件
  • 按‘空格’键快速预览文件；

Microsoft To Do TodoList

  • 重度使用软件；
  • 制定计划必备软件；

由于微软的同步速度太慢了，Microsoft To Do仅仅适合在一个设备的本地时候，如果是在电脑是写下任务，手机上很长时间不能同步任务，多端体验不好！

现在使用Todo List 作为替代软件，同步体验非常棒！

ManicTime

  • 记录电脑使用时间，记录时间都花费在了什么地方，便于自省；
